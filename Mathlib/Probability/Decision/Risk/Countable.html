    Risk in countable spaces #

    In countable spaces, we can write integrals as sums, hence we can write the average or Bayes risk with sums instead of integrals.

    source
    theorem ProbabilityTheory.avgRisk_countable {Θ : Type u_1} {𝓧 : Type u_3} {𝓨 : Type u_5} {mΘ : MeasurableSpace Θ} {m𝓧 : MeasurableSpace 𝓧} {m𝓨 : MeasurableSpace 𝓨} {ℓ : Θ → 𝓨 → ENNReal} {P : Kernel Θ 𝓧} {κ : Kernel 𝓧 𝓨} {π : MeasureTheory.Measure Θ} [Countable Θ] [MeasurableSingletonClass Θ] :
    avgRisk ℓ P κ π = ∑' (θ : Θ), (∫⁻ (y : 𝓨), ℓ θ y ∂(κ.comp P) θ) * π {θ}
