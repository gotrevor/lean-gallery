    Determinant Formula for Generalized Continued Fraction #

    We derive the so-called determinant formula for GenContFract: Aₙ * Bₙ₊₁ - Bₙ * Aₙ₊₁ = (-a₀) * (-a₁) * .. * (-aₙ).

    theorem GenContFract.determinant {K : Type u_1} [Field K] {g : GenContFract K} {n : ℕ} :
    g.nums n * g.dens (n + 1) - g.dens n * g.nums (n + 1) = ∏ i ∈ Finset.range (n + 1), -(g.partNums.get? i).getD 0
