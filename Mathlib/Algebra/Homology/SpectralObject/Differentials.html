    Differentials of a spectral object #

    Let X be a spectral object in an abelian category C indexed by a category ι. In this file, we construct the differentials d : E^{n}(f₃, f₄, f₅) ⟶ E^{n+1}(f₁, f₂, f₃) that are attached to families of five composable morphisms f₁, f₂, f₃, f₄, f₅ in ι. We show that d ≫ d = 0. The homology of these differentials is computed in the file Mathlib/Algebra/Homology/SpectralObject/Homology.lean.

    References #

    • [Jean-Louis Verdier, Des catégories dérivées des catégories abéliennes, II.4][verdier1996]
