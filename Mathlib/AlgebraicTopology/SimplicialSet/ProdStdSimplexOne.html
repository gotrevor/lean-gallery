    source
    noncomputable def SSet.prodStdSimplex.nonDegenerateEquiv₁ {p : ℕ} :
    Fin (p + 1) ≃ ↑((CategoryTheory.MonoidalCategoryStruct.tensorObj (stdSimplex.obj { len := p }) (stdSimplex.obj { len := 1 })).nonDegenerate (p + 1))

    This is an enumeration of the p + 1 nondegenerate dimension-(p + 1) simplices of Δ[p] ⊗ Δ[1]. It sends i : Fin (p + 1) to the nondegenerate simplex consisting of the vertices (0, 0) ≤ (1,0) ≤ ... ≤ (i, 0) ≤ (i, 1) ≤ ... ≤ (p, 1).
