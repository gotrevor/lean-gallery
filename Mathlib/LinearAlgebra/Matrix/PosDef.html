    Positive Definite Matrices #

    This file defines positive (semi)definite matrices and connects the notion to positive definiteness of quadratic forms. In Mathlib/Analysis/Matrix/Order.lean, positive semi-definiteness is used to define the partial order on matrices on ℝ or ℂ.

    Main definitions #

    • Matrix.PosSemidef : a matrix M : Matrix n n R is positive semidefinite if it is Hermitian and xᴴMx is nonnegative for all x.
    • Matrix.PosDef : a matrix M : Matrix n n R is positive definite if it is Hermitian and xᴴMx is greater than zero for all nonzero x.

    Main results #

    • Matrix.PosSemidef.fromBlocks₁₁ and Matrix.PosSemidef.fromBlocks₂₂: If a matrix A is positive definite, then [A B; Bᴴ D] is positive semidefinite if and only if D - Bᴴ A⁻¹ B is positive semidefinite.
    • Matrix.PosDef.isUnit: A positive definite matrix in a field is invertible.
