    def IsAlmostIntegral (R : Type u_1) {S : Type u_2} [CommRing R] [CommRing S] [Algebra R S] (s : S) :
    Prop

    An element s in an R-algebra is almost integral if there exists r ∈ R⁰ such that r • s ^ n ∈ R for all n.
