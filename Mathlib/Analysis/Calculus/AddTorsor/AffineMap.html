    Smooth affine maps #

    This file contains results about smoothness of affine maps.

    Main results #

    • ContinuousAffineMap.contDiff: a continuous affine map is smooth.
    • AffineMap.contDiff_lineMap_uncurry: AffineMap.lineMap is smooth in its three arguments, jointly and pointwise.
