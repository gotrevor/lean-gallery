    The digamma function #

    This file defines the digamma function as the logarithmic derivative of the Gamma function and proves some basic properties.

    Main definitions #

    • Complex.digamma: The digamma function of a complex variable.

    Main statements #

    • Complex.digamma_apply_add_one: The digamma function satisfies the functional equation digamma (s + 1) = digamma s + s⁻¹.
    • Complex.meromorphic_digamma: The digamma function is meromorphic.
