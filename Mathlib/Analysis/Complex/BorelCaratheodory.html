    Borel-Carathéodory theorem #

    This file proves the Borel-Carathéodory theorem: for any function f analytic on the open ball |z| < R such that Re(f z) < M for all |z| < R, we have ‖f z‖ ≤ 2 * M * ‖z‖ / (R - ‖z‖) + ‖f 0‖ * (R + ‖z‖) / (R - ‖z‖)

    Main results #

    • Complex.borelCaratheodory_zero: The theorem under the assumption f 0 = 0
    • Complex.borelCaratheodory: The general version of the theorem

    Implementation Notes #

    The proof applies the Schwarz lemma to the transformed function w z := f z / (2 * M - f z), which maps the ball |z| < R into the unit disk provided that (f z).re ≤ M for all |z| < R. After obtaining bounds on w, we invert the transformation to recover bounds on f.
