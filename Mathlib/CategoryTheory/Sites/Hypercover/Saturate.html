    A relation on a pre-0-hypercover is a commutative diagram

    obj ----> E.X i
 |         |
 |         |
 v         v
E.X j ---> S

      The object.

    • fst : self.obj ⟶ E.X i

      The first projection.

    • snd : self.obj ⟶ E.X j

      The second projection.

    • w : CategoryStruct.comp self.fst (E.f i) = CategoryStruct.comp self.snd (E.f j)
