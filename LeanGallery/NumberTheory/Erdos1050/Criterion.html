    The integer-approximation irrationality criterion #

    The single reusable lemma the whole proof feeds: a real x is irrational if there are integer sequences aₙ, bₙ with bₙ·x − aₙ ≠ 0 for all n and bₙ·x − aₙ → 0. (Hardy–Wright Thm 186 / Van Assche Lemma 5.1.) The standard one-line argument: if x = p/q were rational, then bₙ·x − aₙ = (bₙ·p − aₙ·q)/q is a nonzero rational of absolute value ≥ 1/q, contradicting → 0.

    ⚠️ Check whether mathlib already has this (it has Liouville machinery in Mathlib.NumberTheory.Liouville.* and Irrational API). If a close form exists, delegate to it; this file is a ~20-line standalone fallback otherwise.

    Irrational x

    Irrationality from integer approximations with nonzero, vanishing error. If b n * x - a n ≠ 0 for all n and (fun n => b n * x - a n) → 0, then x is irrational.

    Proof: if x = r were rational with denominator d = r.den, then d·(bₙ·x − aₙ) = bₙ·r.num − d·aₙ is a nonzero integer, hence has absolute value ≥ 1, so |bₙ·x − aₙ| ≥ 1/d > 0 for all n — contradicting bₙ·x − aₙ → 0.
