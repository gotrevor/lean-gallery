# #482 development archive 🗂️

Session handoffs, pending-work lists, on-line research findings, and a reflection, from the original
`gotrevor/erdos-482` development. Kept verbatim.

**These are a record, not documentation.** Their value is largely independent of #482: they are a
worked example of *running a long formalization across many LLM sessions* — what a handoff has to
carry, how a research question gets farmed out and answered, which plans survived contact and which
were abandoned. `findings/` holds the on-line research digests that settled specific literature
questions; `handoff/` holds the session-to-session batons.

If you want to know what was actually proved, read [`../README.md`](../README.md). If you want the
mathematics of the errata, read [`../STOLL-PAIR5-ERRATUM.md`](../STOLL-PAIR5-ERRATUM.md) and
[`../notes/ST06-THM31-ERRATUM.md`](../notes/ST06-THM31-ERRATUM.md). These notes describe intermediate
states that are, by design, wrong or superseded by the end.

## ⚠️ Paths and one filename refer to the pre-gallery layout

Written against the standalone repo, so:

| In these notes | Now |
|---|---|
| `src/Erdos482/…` | [`LeanGallery/NumberTheory/Erdos482/`](../../../LeanGallery/NumberTheory/Erdos482) |
| `papers/SOURCES.md` | [`../SOURCES.md`](../SOURCES.md) |
| `NOTES-FOR-STOLL.md` | [`../NOTES-ON-STOLL-2010.md`](../NOTES-ON-STOLL-2010.md) |

Left uncorrected on purpose. Rewriting them would make the record claim a layout, and a filename,
that did not exist when the note was written.

The rename is worth one line of explanation: that document is a set of notes *about* T. Stoll's 2010
paper — an erratum writeup, the same genre as any other. The old name read as though it were
correspondence addressed to him, which it never was, so it now says what it is.
