      def Plausible.Nat.shrink (n : Nat) :
      List Nat

      Nat.shrink' n creates a list of smaller natural numbers by successively dividing n by 2 . For example, Nat.shrink 5 = [2, 1, 0].

      Equations
      • Plausible.Nat.shrink n = if 0 < n then have m := n / 2; m :: Plausible.Nat.shrink m else []
